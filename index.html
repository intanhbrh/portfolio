Project Overview

SafePoint is a comprehensive emergency response mobile application designed to enhance personal safety through technology. The app provides users with quick access to emergency services, incident reporting capabilities, and vital safety resources all in one place.

Key Features

  • One-touch emergency alert system with GPS location sharing
  • Real-time incident reporting with photo and description uploads
  • Safety resources including emergency contacts and first aid guides
  • User authentication and profile management via Firebase
  • Push notifications for emergency alerts
  • Interactive map showing nearby incidents and safe zones

Technologies Used

Built with Java and Android Studio, utilizing Firebase for real-time database management, authentication, and cloud storage. The app integrates Google Maps API for location services and implements Material Design principles for an intuitive user interface.

Development Process

This Final Year Project involved comprehensive requirements gathering, system design, prototyping, and iterative development. The project emphasizes user-centered design with a focus on accessibility and ease of use during emergency situations.

Project Overview

DeliTEN is a full-stack web-based food ordering system developed specifically for the UNITEN campus community. The platform connects students with campus food vendors, streamlining the ordering process and reducing wait times.

Key Features

  • User registration and authentication system with role-based access
  • Browse menu items with detailed descriptions and pricing
  • Shopping cart functionality with order customization
  • Real-time order tracking and status updates
  • Vendor dashboard for managing menu items and orders
  • Admin panel for system management and analytics
  • Responsive design for mobile and desktop access

Technologies Used

Developed using PHP for server-side logic, MySQL for database management, and modern HTML/CSS/JavaScript for the front-end. The application follows MVC architecture and implements secure session management and SQL injection prevention.

Impact

This Diploma Final Year Project aimed to solve real campus challenges by reducing food ordering queues and improving the overall dining experience for UNITEN students. The system was designed with scalability in mind to accommodate future campus expansion.
